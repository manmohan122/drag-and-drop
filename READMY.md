# 📝 Drag and drop To-Do List Web App

## 📌 Description
This is a simple and interactive To-Do List Web Application built using HTML, CSS, and JavaScript. The application allows users to create, edit, and manage their daily tasks in an organized way. Tasks can be moved between different sections such as ToDo, Working, and Done using a dropdown selector.

The app provides a clean and user-friendly interface, making task management easy and efficient for users.

---

## 🚀 Features
- ➕ Add new tasks dynamically  
- ✏️ Edit tasks directly (content editable)  
- ❌ Auto-delete empty tasks  
- 🔄 Move tasks between sections (ToDo, Working, Done)  
- 🎯 Simple and clean UI design  
- 📱 Responsive layout using Flexbox  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript (DOM Manipulation & Event Handling)
